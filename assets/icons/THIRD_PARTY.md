# Third-Party Icon Assets

This project uses [Tabler Icons](https://tabler.io/icons) for UI/status/tray fallback assets in block 2.7.

- Pack: Tabler Icons (outline)
- License: MIT
- Source repository: https://github.com/tabler/tabler-icons
- Imported for files in:
  - `assets/icons/ui/*.svg`
  - `assets/icons/status/*.svg`
  - `assets/icons/tray/tray-*.svg` (source for generated PNG/ICO)

The primary application icon in `assets/icons/app/*` remains in-house and is not sourced from Tabler.
